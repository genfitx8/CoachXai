# SwingNote

A golf lesson management and swing analysis platform.

## Features

- User authentication (Client/Coach/Admin roles)
- Lesson management and scheduling
- Swing video analysis with AI feedback
- Homework assignments and tracking
- Point-based reward system
- Multi-language support (Korean/English)
- Video comparison tools
- Coach search and booking

## Tech Stack

- React + TypeScript
- Firebase (Auth, Firestore, Storage)
- Vite
- Tailwind CSS
- Google Gemini AI

## Getting Started

1. Install dependencies: `npm install`
2. Configure Firebase credentials in `.env`
3. Run development server: `npm run dev`
